About "Absorption"

Absorption is a game created for the 24th Ludum Dare game competition within less than 48 hours.

Nerd facts:

  • - Written in CoffeeScript, a language that compiles to JavaScript
  • - Does not use any graphics libraries, everything written from scratch
  • - Uses the following libraries: jQuery, jwerty, underscore.js, SoundManager 2
  • - It is open source and available on github.com.

All graphics content has been created using Photoshop CS5.

All audio content has been created using Audacity and Ableton Live.
